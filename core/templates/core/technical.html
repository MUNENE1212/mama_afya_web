Machine Learning Model

How Our AI Predicts Risk

MamaGuardian AI uses supervised machine learning to analyze maternal health data and identify high-risk pregnancies before complications arise. Our model has been trained on comprehensive datasets including:

  • Kenya Demographic and Health Survey (KDHS) data
  • Kaggle's Maternal Health Risk dataset
  • Anonymized patient records from partner clinics

The system analyzes key indicators such as blood pressure, age, previous pregnancy complications, hemoglobin levels, and geographical factors to generate a risk score between 0-100%, classifying each pregnancy as Low, Medium, or High risk.

Accuracy

Our model achieves 90% accuracy in predicting high-risk pregnancies, with sensitivity and specificity values exceeding WHO benchmarks for maternal risk assessment tools.

Offline Capability

Built for rural Kenya, the application works offline and syncs when internet connectivity is available, ensuring continuity in remote areas.

Real-time Alerts

The system generates automatic SMS alerts to both patients and community health workers when risk levels change significantly.

Mobile App & Dashboard

Our solution includes two key interfaces: a mobile application for data collection and monitoring, and a web dashboard for healthcare administrators.

Key Features

User-Friendly Data Entry

Simple forms designed for CHVs with varying levels of technical expertise, supporting multiple languages including Swahili.

Risk Visualization

Color-coded risk indicators and trend graphs show changes over time, making it easy to spot deteriorating conditions.

Resource Allocation Tools

Heat maps highlight areas with high concentrations of at-risk mothers, helping county health teams allocate ambulances and staff effectively.

System Architecture

Our system architecture follows a client-server model with secure data storage and processing:

  • Frontend: React Native mobile app and React web dashboard
  • Backend: FastAPI server for API endpoints and ML model hosting
  • Database: MongoDB for storing patient data with end-to-end encryption
  • SMS Gateway: Integration with Africa's Talking API for alerts
  • ML Pipeline: Python-based data processing with scikit-learn and XGBoost

Pilot Program

We are currently piloting MamaGuardian AI in two counties in Kenya:

  • Vihiga County: Working with 50 CHVs and 5 health facilities
  • Turkana County: Partnering with 30 CHVs and 3 nomadic health clinics

Initial results show a 35% improvement in early referrals for high-risk cases and a 28% reduction in emergency transport times when the system identifies urgent cases.
